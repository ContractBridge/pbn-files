(http://www.okbridge.com) with more than 2000 games.
In each game you can enjoy the playing of Benito Garozzo (papi).

The data have been supplied by George Mavridis
(gm@mavridis.tng.oche.de).

All files satisfy the PBN 2.0 format:
Each deal has a score table.
